แก้สำหรับ Vercel npm install error:
- ลบ package-lock.json ออก เพื่อไม่ให้ Vercel ใช้ lockfile ที่มี registry path จากเครื่องสร้างไฟล์
- ลบ dependency ที่ยังไม่ใช้ตอนนี้: openai, @supabase/supabase-js
- เพิ่ม .npmrc บังคับใช้ public npm registry
- ยังเป็น planner/dashboard localStorage mode เหมือนเดิม

วิธีอัปโหลด GitHub:
1) ลบไฟล์เก่าใน repo ให้สะอาดก่อน
2) อัปโหลด app, components, lib และไฟล์ root ทั้งหมดจากโฟลเดอร์นี้
3) ห้ามอัป package-lock.json เพราะเวอร์ชันนี้ตั้งใจไม่มี
4) ใน Vercel ใช้ Install Command: npm install --no-audit --no-fund
